# py2app-client

Simple app that allows to create a MacOS application from Python script.

## Requirement

* MacOS
* Python > 2.7

## Get started

#### Using pip
```
pip install -U py2app
```
#### Using easy-install
```
easy_install -U py2app
```

## Features

* Graphic app
* Bundle settings
* Icon setting
